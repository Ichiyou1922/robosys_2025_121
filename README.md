# plusコマンド
![test](https://github.com/Ichiyou1922/robosys2025/actions/workflows/test.yml/badge.svg)
## プログラムの内容  
- 標準入力から入力された数値を和算する．
## 使い方
- ダウンロード方法
  - $ git clone git@github.com:Ichiyou1922/robosys2025.git
- 使い方
  - 標準入力から幾つかの整数をプログラムに送る．
  - 例: $ seq 5 | ./plus
## 必要なソフトウェア
- Python
  - テスト済みバージョン: 3.7~3.10

## テスト環境
- Ubuntu 24.04.5 LTS

## 実行できない場合
- $ chmod +x plus で権限を与える．
